## Platforms
- not all platforms need to be technology platforms
- two sided (digital) platforms --> airbnb, facebook, amazon, uber... 
- economics of platforms vs 2-sided platforms
	- platforms:
		- value creation AND value capture via scale and scope economies
	- 2-side platforms
		- value creation for both sides of the market via network economies
		- value capture: via switching costs (expensive/costly to leave the platform)
