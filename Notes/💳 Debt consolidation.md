
Snowball method on debt consolidation:


| Account              | Interest rate | Minimum payment | Balance         |
| -------------------- | ------------- | --------------- | --------------- |
| Fidelity Credit Card | 17.49%        | ~$400           | $11,378.55      |
| JetBlue Credit Card  | 29.74%        | ~$300           | $5,034.16       |
| Volvo loan           | 5.49%         | $1,173.99       | $32,680.53      |
| Audi loan            | 5.25%         | $1,057.39       | $31,307.48      |
| Pottery Barn credit  |               |                 |                 |
| Apple Credit Card    | 22.49%        | $185            | $1,286.90       |
| HELOC                | 3.5%          | $500            | $85,842.51      |
| Petro AC Loan        | 0%            | $345            | $9,167.05       |
| Jordan's credit      |               |                 |                 |
| **TOTAL**            |               | **$4,000**      | **$176,697.18** |

In this method we go highest interest rate to lowest. We pay minimum on everything except the one that we want to prioritize; when that's paid, roll the minimum from that into the next one and so on. The order for us needs to be:

JetBlue -> Apple -> Fidelity -> Volvo -> Audi -> Petro

We can speed this up with tax refunds/bonuses/etc, but the idea would be to not use some of those credit cards and credit lines as we go, to keep debt managed.
